the problem

You only needed an app like this to pre-order the album; after that, the app is useless.

the goal

We aim to create a community where users can communicate about the band inside the app. In this way, they can keep using the app.

understand the users

The user research is mainly based on interviews. We need to understand better who the users are and how the product can meet the user's needs. We need to keep the user on the app not only for the primary function but also need to find an added value.

the design

The general idea is to have a page with very few elements. Although the main components should be more visible, I like having a full background with the cover album when possible.
